# Astrodex - Backend 🌌

Serveur Node.js / Express

## 🛠️ Installation
1. Clonez le dépôt sur la branche dev :
   `git clone -b dev [URL_DU_REPO]`
2. Installez les dépendances :
   `yarn install`

## 🔑 Configuration (.env)
Créez un fichier `.env` à la racine et ajoutez la variable suivante (donné par Allan sur Orion) :
`MONGODB_URI=votre_lien_mongodb`

## 🚀 Lancement
`yarn dev` (utilise nodemon)
